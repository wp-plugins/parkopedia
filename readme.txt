== Description ==

The Parkopedia plugin let's you embed an interactive parking map onto your website, helping your users find parking around your location.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `parkopedia.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Embed your parking map using our shortcode: [parkopedia lat="51.5078238802915" lng="-0.09417821970849796" width="800" height="500" title="London Parking"]
